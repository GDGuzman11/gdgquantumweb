## 🧠 Built Entirely From Scratch

No templates. No shortcuts. This landing page was handcrafted with HTML, CSS, and JavaScript — featuring:

- 🧠 **Custom Canvas Neuron Network** (inspired by Ben Scott's site)
- ✍️ **Typing effect** with animated cursor and responsive font scaling
- 🕹️ **CTA Button** that fades in after the intro
- 🌐 **Responsive Design** — clean on desktop and mobile
- 🎨 **Cream-toned modern aesthetic** with matte black highlights
- ⚡ Zero frameworks — pure vanilla for total control

---

## 🔧 Tech Stack

| Tech | Description |
|------|-------------|
| `HTML5` | Structure and semantic layout |
| `CSS3`  | Clamp-based responsive styling with animated effects |
| `JavaScript (ES6)` | Typing, canvas motion, and modular neurons |
| `Canvas API` | Custom neuron render + interaction logic |
| `Git + GitHub` | Version control and collaboration |
| `GitHub Pages` | (Optional) live site hosting |
