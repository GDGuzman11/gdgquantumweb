## 🌊 Neural Animation System

The website features a sophisticated dual-layer neural network visualization:

### Canvas Layer (neural-canvas.js)
🧠 **Floating Particles**: Physics-based movement with boundary collision  
🔗 **Dynamic Connections**: Lines drawn between nearby particles  
⚡ **Real-time Rendering**: 60fps canvas animation loop  

### CSS Layer (neural-flow.js)
🌊 **Data Streams**: Animated flowing data with warm color variants  
💫 **Floating Nodes**: Gentle floating animation with glow effects  
💥 **Pulse Waves**: Expanding circles that trigger code generation  
💻 **Code Snippets**: AI-themed code appears and fades dynamically  

### Border Animations (Neural Border Card)
🎯 **Falling Nodes**: Particles fall and connect to form border segments  
⚡ **Connection Pulses**: Energy travels between connected nodes  
🖼️ **Border Drawing**: Progressive line drawing as connections form

## 🔥 The Glitch Effect

The centerpiece animation transforms the hero text:

```
"we make the impossible." → "we make the inevitable."
```

**Technical Implementation**:
- CSS keyframe animations with glitch distortion effects
- RGB color separation with chromatic aberration
- Scale and blur transformations
- Precise timing for dissolve/materialize sequence
- Mobile-optimized reduced intensity

**Timing**:
- 3 second initial delay
- 1.5 second dissolve animation
- 1.5 second materialize animation
- 8 second cycle repeat

## 🛡️ Security & Protection

### **Contact Form Security**
✅ **Input Sanitization** — Removes scripts, HTML tags, dangerous protocols  
✅ **Pattern Detection** — Blocks malicious keywords and SQL injection attempts  
✅ **Rate Limiting** — Max 3 submission attempts per minute  
✅ **Honeypot Protection** — Hidden field to catch automated spam  
✅ **Content Validation** — Real-time checking with visual feedback  
✅ **Character Limits** — Prevents buffer overflow attempts  

### **Security Patterns Blocked**
```javascript
const blockedPatterns = [
  /\b(hack|exploit|injection|malware)\b/i,
  /\b(script|iframe|object|embed)\b/i,
  /(union\s+select|drop\s+table)/i,
  /(data:|vbscript:|javascript:)/i
];
```

### **Performance & Accessibility**
✅ **Error Handling** — Global error catching and logging  
✅ **Performance Monitoring** — Load time and paint metrics  
✅ **Reduced Motion** — Accessibility support for motion sensitivity  
✅ **Mobile Optimization** — Lighter animations, responsive design  
✅ **SEO Optimized** — Semantic HTML, meta tags, structured data
